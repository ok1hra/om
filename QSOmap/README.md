This program will read an ADIF file and map the QSOs on a map of the earth

You will need to download a "Blue Marble" image for the background. NASA Blue Marble images are in the correct projection and aspect ratios.
You can download images here: https://visibleearth.nasa.gov/collection/1484/blue-marble

By default, it will draw worked grids, DX station locations, QSOs paths (short path), and your loaction.
You can comment out the specific drawing functings in the main function if you don't want to draw everything
